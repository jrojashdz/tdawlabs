En CSS, los márgenes contiguos de dos o más cajas (que pueden ser o no ser hermanos) pueden combinarse para formar un único margen. Los márgenes que se combinan de esta manera se dice que se colapsan, y el margen combinado resultante se llama margen colapsado.
